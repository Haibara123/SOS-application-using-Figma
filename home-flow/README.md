# Home & Core Features – SOS Mobile App

This flow presents the main features of the SOS application.

## Main Functions

### SOS Button
- One-tap SOS: sends emergency signal after 3 seconds
- Hold SOS: select rescue type and upload evidence
- SOS voice recording

### Notifications
- View rescue team information
- Receive disaster and weather alerts

### Community
- Administrator posts
- Donations from sponsors (rice, money, supplies)

### Contacts
- Emergency numbers: 112, 113, 114
- Saved personal emergency contacts
- Chat with administrator

### Settings
- View personal information
- SOS history
- Support & help information

**Tool:** Figma
